À propos




Online test est une application web de passation des tests en ligne.
Chaque test est composé de plusieurs questions qui peuvent être une question à choix multiples (QCM) ou une question de codage.
Chaque étudiant peut utiliser cette application pour passer ces test et aussi pour consulter ces scores.
Les scores sont communiqués à l’application GestNotes qui s’occupe de la gestion des notes.
Les professeurs utilisent l’application pour créer et poster les tests.
Pour utiliser cette application, il faut tout d’abord s’authentifier.
Les administrateurs sont responsables de la gestion des comptes, ils peuvent consulter les scores.

Un Questionnaire à Choix Multiples (QCM) est une catégorie d’exercices, de tests, se présentant sous la forme d’une question/affirmation suivie de plusieurs propositions de réponses, parmi lesquelles se trouve(nt) une ou plusieurs réponse(s) correcte(s).
